Printer Setup with QZ Tray

Client-side printing for your POS system
How Printing Works

This POS system uses QZ Tray, a client-side application, to connect to your local thermal printers. This allows you to print receipts directly from your browser, no matter where you are.

Setup Instructions:
  1. Install QZ Tray: If you haven't already, download and install the QZ Tray application on every computer that will be used for printing. You can download it from the official QZ Tray website.
  2. Run QZ Tray: Make sure the QZ Tray application is running in the background on your computer before you start making sales. You should see its icon in your system tray.
  3. Connect your Printer: Ensure your thermal receipt printer is properly connected to your computer (via USB, Bluetooth, etc.) and is installed correctly in your operating system.
  4. Start Printing: When you complete a sale, the system will automatically try to find a suitable receipt printer and send the print job. You may see a confirmation popup from QZ Tray the first time you print.
No Server-Side Configuration Needed: With this method, you no longer need to configure printer IP addresses in the settings. Printer selection is handled locally on each machine.
